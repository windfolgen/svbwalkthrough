# Workflow Logs Directory

This directory contains execution log files generated by the bootstrap run scripts (`run.wl`).
Each run script automatically redirects its console printouts and warning/error messages here.

## Log Files
Files are named as `<label>.log` (e.g., `threeloophard2.log`).
You can inspect these files in real-time during long calculations to monitor progress and locate which stage or step is currently executing.
